# Supervised Learning

Learn to predict labels from labeled data. The most intuitive introduction to ML.

---

## Start Here

### [1. Linear Regression](./linear-regression/notes.md) →
Predict continuous values. Learn the fundamentals: loss functions, gradient descent, hyperparameters.

### [2. Logistic Regression](./logistic-regression/notes.md) →
Binary classification using probability. Sigmoid function, log loss, decision boundaries.

### [3. Classification Metrics](./classification/notes.md) →
Evaluate classification models. Confusion matrix, precision, recall, F1-score, ROC-AUC.

---

## Learning Flow

```
Linear Regression → Logistic Regression → Classification Metrics
```

Each topic builds on the previous one. Complete them in order.

---

## Quick Reference

| Problem Type | Algorithm | Key Concept |
|--------------|-----------|-------------|
| Predict a number | [Linear Regression](./linear-regression/notes.md) | MSE, gradient descent |
| Predict a category (2 classes) | [Logistic Regression](./logistic-regression/notes.md) | Sigmoid, probability |
| Measure classifier quality | [Classification](./classification/notes.md) | Precision, recall, F1 |
