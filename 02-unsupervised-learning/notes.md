# Unsupervised Learning

Discover patterns in unlabeled data. No labels needed — find structure instead.

---

## Topics

### [Clustering](./clustering/notes.md)
Group similar data points together. K-Means, Hierarchical, DBSCAN.

### [Dimensionality Reduction](./dimensionality-reduction/notes.md)
Reduce features while preserving information. PCA, t-SNE, UMAP.

### [Anomaly Detection](./anomaly-detection/notes.md)
Identify outliers and unusual patterns. Gaussian methods, Isolation Forest.

---

## Learning Flow

```
Clustering → Dimensionality Reduction → Anomaly Detection
```

## Quick Reference

| Task | Algorithm | Best For |
