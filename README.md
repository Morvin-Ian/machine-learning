## How to Use This Course

1. **Start with Supervised Learning** — builds intuition with clear, labeled examples
2. **Progress to Unsupervised Learning** — discover patterns without labels
3. **Dive into Deep Dives** when you want to understand the math
4. **Explore Advanced Concepts** for modern techniques
5. **Learn Real World ML** before deploying to production

Each section includes theory and runnable Python examples.
